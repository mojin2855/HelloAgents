# HelloAgents 学习项目

基于 [hello-agents](https://github.com/datawhalechina/hello-agents) 仓库的 Agent 学习与代码练习项目。

## 环境要求

- Python >= 3.10
- Poetry（包管理工具）

## 快速开始

```bash
# 安装依赖
poetry install

# 进入虚拟环境
poetry shell

# 运行代码
python chapters/chapter4/hello_agent.py
```

## 项目结构

```
HelloAgents/
├── chapters/          # 各章节学习代码
├── data/              # 数据文件
├── config/            # 配置文件
├── pyproject.toml     # Poetry 项目配置
└── README.md
```

## 学习路线

参考 hello-agents 教程，从 Agent 基础概念到多 Agent 应用开发。
